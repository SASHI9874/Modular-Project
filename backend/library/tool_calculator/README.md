# Calculator Tool

Safe mathematical expression evaluator.

## Features

- Basic operations: +, -, *, /, //, %, **
- Functions: abs, round, min, max, sqrt, sin, cos, tan
- Safe evaluation (no exec/eval)

## Usage

**Input:** `expression` (string)
**Output:** `result` (number)

## Examples
```python
"5 + 3"        → 8
"25 * 4"       → 100
"2 ** 10"      → 1024
"sqrt(16)"     → 4.0
"(10 + 5) / 3" → 5.0
```
