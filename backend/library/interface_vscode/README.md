# VS Code Interface

VS Code extension for AI coding agents.

## Features

- Chat panel for agent interaction
- Explain selected code
- Refactor code suggestions
- Real-time WebSocket connection

## Installation

In downloaded app:
```bash
cd vscode-extension
npm install
npm run compile
code --install-extension .
```

## Usage

1. Start backend: `cd backend && python main.py`
2. Open VS Code
3. Click AI Agent icon in sidebar
4. Start chatting!
