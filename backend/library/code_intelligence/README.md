# Code Intelligence

Codebase understanding for coding agents.

## Operations

- **index** - Index workspace files
- **search** - Find relevant files by query
- **get_context** - Get context for a file
- **analyze_deps** - Find file dependencies

## V1 Features

- Simple file indexing
- Keyword-based search
- Dependency detection

## Future (V2)

- Tree-sitter parsing
- Embeddings + vector search
- AST analysis
